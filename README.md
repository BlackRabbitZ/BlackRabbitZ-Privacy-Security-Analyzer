# Overview

BlackRabbitZ Privacy & Security Analyzer is a graphical Python-based awareness and educational tool focused on:

- Privacy
- Security
- Anonymity
- Tracking exposure
- Digital behavior awareness

The tool helps users understand:

- how much data they expose
- how insecure settings affect them
- how tracking and telemetry work
- how to improve privacy and security

---

# Features

## PC / Laptop Analysis
- Operating system analysis
- Encryption awareness
- Update behavior
- Admin rights checks
- Telemetry awareness
- Firewall & backup awareness

## Smartphone Analysis
- Android / iPhone analysis
- App permissions
- Tracking settings
- Location settings
- Bluetooth/NFC awareness
- Security update awareness

## Browser & Internet
- Browser privacy
- Tracking protection
- VPN / Tor awareness
- Public WiFi risks
- Dangerous downloads

## Account Security
- Password manager awareness
- Password reuse detection
- 2FA awareness
- Cloud dependency

## Scores
- Security Score
- Privacy Score
- Anonymity Score
- Tracking Exposure
- Attack Surface

## Learning Mode
Integrated explanations for:
- privacy
- security
- metadata
- tracking
- VPN
- Tor
- anonymity

# Installation & Start (Windows)

# 1. Install Python

Download Python 3.10 or newer:

https://www.python.org/downloads/

Important during installation:

✅ Enable **"Add Python to PATH"**

Then install Python normally.

---

# 2. Download Project

```bash
git clone https://github.com/YOURNAME/BlackRabbitZ-Privacy-Security-Analyzer.git
```

Or download ZIP directly from GitHub.

---

# 3. Start Application

```bash
python BlackRabbitZ_Privacy_Security_Analyzer_EN.py
```

---

# Installation & Start (Linux)

## Debian / Ubuntu

```bash
sudo apt update
sudo apt install python3 python3-pip
```

Start application:

```bash
python3 BlackRabbitZ_Privacy_Security_Analyzer_EN.py
```

---

## Arch Linux

```bash
sudo pacman -S python python-pip
```

Start:

```bash
python3 BlackRabbitZ_Privacy_Security_Analyzer_EN.py
```
